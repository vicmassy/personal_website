Sobre Mi

Hola! Soy Victor Massagué Respall

Soy un ingeniero de software de robótica apasionado de los drones. Mi área de interés cae en la intersección entre inteligencia artificial y robótica. Estoy focalizado en diseñar algoritmos de planificación de rutas que mejoran las habilidades y seguridad de los robots móviles.

Mi objetivo ambicioso es diseñar software para robots autónomos que nos ayudarán a explorar el espacio.

Durante mi tiempo libre normalmente juego al fútbol, escucho música, cocino algo bueno y quedo con amigos.

Selección de Proyectos

El Papel de la Modularidad y la Neurorregulación para la Producción de Comportamientos Múltiples

El Papel de la Modularidad y la Neurorregulación para la Producción de Comportamientos Múltiples

Abril 2020 - Mayo 2020 | Innopolis University, Innopolis, Rusia

El proyecto implica la implementación de redes reguladoras de este tipo y la realización de experimentos que implican la producción de diferentes comportamientos. El proyecto se desarrolla con algoritmos evolutivos y entornos PyBullet.

Modelado Geométrico y de Rigidez y Diseño de Experimentos de Calibración para el Manipulador de 7 DOF

Detección de Postura Política en Tweets en Catalán y Español

Detección de Postura Política en Tweets en Catalán y Español

Nov 2017 - Dic 2017 | Innopolis University, Innopolis, Rusia

Una técnica sencilla para la detección de posturas con la independencia de los tuits de Cataluña. Se utilizan funciones de inserción de palabras word2vec para esta detección. El sistema ha producido el mejor resultado para tuits españoles en comparación con otros participantes en IberEval 2017.
[pdf | Github]

Robolang

Robolang

Feb 2017 - Jun 2017 | Universidad Politécnica de Cataluña, Barcelona, España

Trabajé con dos compañeros desarrollando un lenguaje de programación para facilitar el proceso de programación de los robots Lego Mindstorms. Además, incluye pedidos predefinidas que simplifican un número significativo de líneas de código. Finalmente, incluye un compilador que envía el código directamente al robot. El proyecto se hizo con Java y ANTLR3.
[pdf | Github]

Document Similarity

Document Similarity

Este proyecto realizado junto con dos compañeros de clase, implementa Jaccard Similarity, Mihash Similarity y Local Sensible Hashing para detectar y calificar la similitud entre documentos con C++.
[pdf | Github]

HeteSim

HeteSim

Feb 2016 – Maig 2016 | Universidad Politécnica de Cataluña, Barcelona, España

HeteSim es un programa para realizar búsquedas relacionales (y normales / simples) en una base de datos compuesta como gráfico, utilizando la medida HeteSim para la relación entre gráficos heterogéneos. La base de datos que se utiliza para la implementación es la colección de nombres de diferentes elementos (de diferentes tipos) de una biblioteca pública de informática.
[Github]

Maqueta de una Casa Domótica

Maqueta de una Casa Domótica

Mayo 2013 – Dic 2013 | INS Molí de la Vila, Capellades, España

Este fue mi proyecto final de secundaria. Consiste en un modelo de 80x50cm de una casa, aplicando diferentes automatizaciones. Por ejemplo, los sensores de las luces del jardín cuando oscurece o una puerta automatizada de apertura / cierre cuando un coche se acerca. Todo el proyecto se programó con Arduino funcionando a bordo de dos placas Arduino UNO.
